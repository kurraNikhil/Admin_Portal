## Objective:

Develop a React-based frontend application with the following features:

1.  A modern and user-friendly admin portal design
2.  Implementation of various React components, including Sign In Page, Register Page, Checkbox, DatePicker, Collapse, Descriptions, Button, Form, Table, Icon, Input, Tabs, Typography, Radio, Tag, Layout, Slider, Timeline, Divider, Switch, Grid, Layout, Transfer, Breadcrumb, TreeSelect, TimePicker, Tooltip, Tour, Notification, Progress, Dropdown, Upload, Result, Menu, Avatar, Spin, Pagination, Badge, Reports, Steps, AutoComplete, Calendar, Card, and Cascader
3.  Utilization of mock data to simulate backend interactions
4.  Assumption of an ExpressJS-based backend
5.  Creative freedom in UI design and color palette selection

## Terminal Commands

1. Download and Install NodeJs LTS version from [NodeJs Official Page](https://nodejs.org/en/download/).
2. Navigate to the root / directory and run yarn install/npm install to install our local dependencies.
3. npm start
